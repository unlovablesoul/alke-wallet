# Alke Wallet

La billetera Alke fue diseñada para la evaluación del módulo 2 "Fundamendos de Front-End"

## Descripción

El proyecto se basa en una aplicación web que simula una billetera digital, la cual permite realizar operaciones básicas como depósito, transferencia y visualización de movimientos.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap
- JavaScript
- jQuery

## Estructura

El proyecto se organiza en archivos HTML independientes para cada vista, junto con carpetas para estilos y scripts.

## Uso

Abrir el archivo index.html en el navegador para iniciar la aplicación. Hacer ingreso con las credenciales admin@alke.cl y contraseña 1234
